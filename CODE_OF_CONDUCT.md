# Code of Conduct

## Our Standard

This project expects respectful, direct, and constructive collaboration.

Examples of behavior that contributes to a healthy community:

- Giving actionable technical feedback
- Assuming good intent while being precise about problems
- Respecting differing experience levels
- Focusing criticism on ideas, code, and behavior

Examples of unacceptable behavior:

- Harassment or personal attacks
- Discriminatory language or conduct
- Trolling, intimidation, or deliberate disruption
- Publishing private information without permission

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, and related community channels.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

## Reporting

For conduct concerns, contact the maintainer privately through GitHub.
